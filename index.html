Setting up machine learning development environment

I decided to write this after facing the surprisingly annoying process of installing various packages for machine learning purposes. Let’s start from the beginning. Create an environment on anaconda: conda create -n pytorch-env python=3 Switch to the pytorch-env environment: conda activate pytorch-env Install PyTorch according to the official documentation. Notice that it includes cpuonly instead of cudatoolkit. This means that there will be no CUDA support, which is intentional because I’m doing this in a VM.
